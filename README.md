# Редактор карт для 2D-платформеров
## Инструкция пользователя
### 1. Установка и запуск:
1.Создайте новый проект в Unity или откройте существующий

2.В окне Project создайте структуру папок (Рис.10)

<img width="340" height="228" alt="image" src="https://github.com/user-attachments/assets/77f2b0e8-93e6-43dd-8482-b22b44cebf8f" />

3.Разместите файлы редактора в соответствующих папках:

MapEditorWindow.cs  =>  Assets/Editor/
LevelData.cs, LevelVisualizer.cs, LevelLoader.cs  => Assets/Scripts/

4.Дождитесь компиляции проекта 

### 2. Создание базовой сцены

1.В меню выберите File  => New Scene

2.Сохраните сцену: File  => Save As  =>  "MainScene" в папке Assets/Scenes/

### 3. Создание управляющих объектов

1.В Hierarchy нажмите правой кнопкой  => Create Empty

2.Назовите объект "LevelManager"

3.Добавьте компоненты:

Add Component  => Level Visualizer
Add Component  => Level Loader

### 4. Создание префабов тайлов

- Создание Ground тайла:

1.Right-click in Hierarchy => 2D Object => Sprite
2.Назовите "GroundTile"
3.В Sprite Renderer установите требующийся цвет
4.Добавьте коллайдер: Add Component => Physics 2D => Box Collider 2D
5.Перетащите объект в папку Assets/Prefabs/Tiles/ для создания префаба
6.Удалите оригинальный объект из сцены

- Создание Wall тайла:

1.Назовите префаб "WallTile"
2.Повторите шаги для Ground, но установите другой цвет

- Создание Coin тайла:

1.Назовите префаб "Coin"
2.Повторите шаги для предыдущих тайлов. В компоненте "BoxCollider2D" поставьте галочку в поле "IsTrigger"

### 5. Активация редактора

1.В главном меню Unity выберите Tools => 2D Map Editor

2.Должно открыться новое окно с заголовком "2D Map Editor"

### 6. Проверка работоспособности

1.В окне редактора нажмите кнопку "New"

2.В консоли должно появиться сообщение "Создан новый уровень..."

3.В папке Assets/Data/ должен появиться файл NewLevel.asset

### 7. Настройка LevelManager

1.В Hierarchy выберите "LevelManager"

2.В компоненте "Level Visualizer":

Перетащите NewLevel.asset в поле "Level Data"

3.В компоненте "Level Loader":

Перетащите соответствующие префабы в поля:
- Ground Tile Prefab => GroundTile.prefab
- Wall Tile Prefab => WallTile.prefab
- Coin Prefab => Coin.prefab
- 
4.Убедитесь, что включены опции Auto Load On Start и Clear Previous Tile
  
5.Установите координату Z объекта Main Camera в значение -10

### 8. Создание уровня

Процесс проектирования:

1.Выберите тип тайла в палитре (Ground, Wall, Coin)
2.Активируйте инструмент рисования (Paint Tool)
3.Размещайте тайлы в окне Scene View кликами или перетаскиванием
4.Используйте инструмент стирания для коррекции

Рекомендации по работе:

1.Начинайте с создания базовой платформы из Ground тайлов
2.Добавляйте стены и препятствия с помощью Wall тайлов
3.Размещайте декоративные элементы через Coin тайлы
4.Регулярно сохраняйте прогресс кнопкой "Save"

### 9. Управление и горячие клавиши

Основные действия:

- P - активация инструмента рисования
- E - активация инструмента стирания
- Левый клик - размещение/удаление тайла
- Движение мыши с зажатой левой кнопкой - непрерывное рисование/стирание
